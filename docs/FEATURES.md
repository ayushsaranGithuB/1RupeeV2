# Features

## User Features

Users can:

- Sign in with Google or Magic Link
- Browse active campaigns
- View campaign details
- Select one or more campaign support tiers
- Top up their wallet
- Pause or resume pledges
- View wallet balance
- View remaining donation runway
- View donation history
- View wallet transactions
- Update name and avatar
- Manage authentication
- Sign out

### Campaign Support Tiers

Every campaign defines between 3 and 5 support tiers.

Example:

| Tier | Daily | Monthly | Impact |
| --- | --: | --: | --- |
| Daily Supporter | ₹1 | ₹30 | Help sustain the cause. |
| Impact Supporter | ₹5 | ₹150 | Provide books for one child each month. |
| School Lunch Sponsor | ₹10 | ₹300 | Fund a student's lunches for a month. |
| Community Champion | ₹35 | ₹1000 | Support a classroom or family. |

Users subscribe to a tier rather than choosing an arbitrary amount.

### Wallet

Users can:

- View balance
- Top up wallet
- View transaction history
- See recharge recommendations

Suggested presets: ₹100, ₹250, ₹500, ₹1000

## Admin Features

Admin can:

- Create, edit, approve, and archive NGOs
- Upload NGO logos
- Store verification notes
- Create, edit, and archive campaigns
- Upload campaign hero images
- Assign NGOs to campaigns
- Configure 3–5 support tiers per campaign
- Edit impact descriptions
- Search and view users
- View user profiles, wallets, pledges, donations
- Manually credit/debit wallets (audit logged)
- Suspend users
- Search donations and filter by NGO/campaign
- Export donation CSV
- View ledger (never edit donation history)
- Generate payout reports
- Review totals and mark payouts complete
- Upload payout receipts
- Download payout history
- Publish transparency reports
- Upload annual and audit reports

### Admin Dashboard

Display:

- Total Users
- Active Donors
- Active Pledges
- Total Donation Volume
- Wallet Balance Across Platform
- Active Campaigns
- Pending NGO Applications

---

## NGO Features (MVP Phase)

During MVP:

- Apply to join
- Submit verification documents
- Request campaign creation
- Receive monthly payouts

No NGO dashboard or self-service portal will be built during MVP.

---

## Out of Scope (MVP)

- Native mobile apps
- Referral program
- Badges
- Social features
- Comments
- Volunteer management
- AI recommendations
- Public APIs
- NGO self-service portal

---

## Definition of Done

The MVP is complete when:

- Users can recharge a wallet
- Users can pledge to campaign tiers
- Daily donations are processed automatically
- Campaign totals update correctly
- Admin can process monthly NGO payouts
- Transparency page reflects live platform statistics
